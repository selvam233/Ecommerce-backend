## Description
This is a **Spotify Clone** built using **React** with functional components and **React Hooks**.  
It replicates the core features of Spotify, allowing users to browse albums, play songs, and enjoy a modern music player interface.  

## Features
- Browse albums and songs
- Play, pause, next, previous song controls
- Shuffle and loop functionalities
- Responsive and interactive UI
- Context API for global state management

## Technologies Used
- **Frontend:** React, React Hooks, CSS
- **State Management:** Context API
- **Bundler/Build Tool:** Vite
- **Audio Player:** HTML5 Audio API
